Encrypted Client Hello and connection reliability.

- Added Encrypted Client Hello (ECH). When your server has it enabled, Bedlam hides the real server name from anyone watching the connection, and there is a new ECH field in the profile editor.
- The status shown in the app now matches the real tunnel. It no longer stays on "Connected" after traffic stops flowing, and it reconnects within a few seconds when you switch between Wi-Fi and cellular.
- When the network drops for a while, Bedlam no longer gives up. It keeps trying and reconnects on its own once the network returns, instead of stopping after a few minutes.
- If the VPN was running, Bedlam restores it after a phone restart.
- Reconnect attempts slow down during a long outage, so Bedlam uses less battery when there is no signal.
- A profile with a stale ECH setting now stops with a clear reason instead of retrying.
- Updated the bundled Hysteria core to 2.10.0.
