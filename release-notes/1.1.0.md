In-app updates and interface polish.

- Bedlam checks GitHub for a new release when it starts and offers the update right in the app: release notes, download progress, and installation in one screen, with the APK matched to your device. "Not now" mutes that version until the next one.
- The VPN notification shows a live connection timer and current up/down speeds at a glance; expanding it reveals session totals, the server address, and UDP relay status.
- The status bar icon is redrawn as a vector — crisp at every density instead of a scaled-down bitmap.
- Profile configuration got a floating action toolbar in place of the expanding corner button, switches sit beside their labels while editing, dividers step aside in edit mode, and the transition between viewing and editing is smoothly animated. The field reference link is now a proper button.
